express后端框架的应用:

1.安装--使用npm安装:
在工程文件夹下快速生成npm项目
*npm init -y
将express安装进入项目依赖
*npm install express --save

2.使用express-generator快速生成工具,快速生成框架
*全局安装npm i express-generator -g
*安装成功后 express --version 可以查看对应的版本号
*输入express [项目名] -e 加入ejs模板引擎的支持
*创建好项目之后进入项目根目录 安装依赖 npm i
*启动项目 npm start
*浏览器中打开3000端口号进入应用
